# Workout Planner & Tracker Pro

Final project: plan workouts by creating plans with exercises, log training sessions, and track progress with stats and charts.

## Features
- Plans: create/rename/delete
- Exercises inside plans: add/rename/delete
- Logs: plan + exercise + weight + reps
- Automatic volume calculation (weight × reps)
- Stats panel + filters (plan/exercise/text)
- Progress chart per exercise (Chart.js)
- Smart suggestion for progressive overload
- Export/Import JSON + Reset tools

## Tech
HTML, CSS, JavaScript, LocalStorage, Chart.js (CDN)

## Run
Open `src/index.html` in your browser.

## Project Structure
- docs/PRD.md — requirements + workflow
- src/index.html — UI
- src/styles.css — styling
- src/app.js — logic, storage, chart, features
