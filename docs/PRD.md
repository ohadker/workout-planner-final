# Smart Workout Planner - PRD

## App Description
A simple web app to log workouts and track progress per exercise.

## Skills from Course
- HTML structure and forms
- CSS styling
- JavaScript DOM + events
- LocalStorage for saving data
- Working with GitHub repo + README + PRD

## MVP Features
- Add workout log (exercise, weight, reps)
- Display history list
- Save data in LocalStorage
- Delete a log

## Resources
- VS Code
- GitHub repository
- Browser

## Workflow
1. Create PRD + README
2. Create file structure (docs/src)
3. Build UI in HTML/CSS
4. Implement logic in JS (add, render, delete)
5. Add LocalStorage persistence
6. Test in browser
7. Upload to GitHub
